CSC152: Proxmox Connection Setup

Proxmox Connection Guide

Welcome!

This guide will walk you through the initial steps to get connected to the course network and access your personal Proxmox Hypervisor. Please follow the steps carefully. This process will also be recorded as part of class on 1/28/2026.

Information Needed Before Starting

Before you begin, please make sure you have the following credentials. You should have received an email with all of the required information.

  • OpenVPN Username
  • OpenVPN Password
  • Private Key Password: abnormal
  • Proxmox IP Address (Your unique URL)
  • Proxmox Username: root
  • Proxmox Password (this will be unique to you)

Setup Instructions

1

Download OpenVPN Connect

Download and install the OpenVPN Connect client for your operating system (Windows, macOS, or Linux).

Download OpenVPN
2

Download VPN Configuration Files

Click the link below to download the VPN configuration files. The zip file contains:

  • OpenVPN.ovpn (for off-campus access)
  • OpenVPN-onCampus.ovpn (for on-campus access)
  • secret (your credential file)
Download Configuration Files

Crucial: After unzipping, you must keep the secret file in the exact same folder as the .ovpn file you intend to use.

3

Add Your OpenVPN Credentials to the 'secret' File

You will be provided with a unique username and password. Open the secret file with a plain text editor (like Notepad on Windows or TextEdit on Mac).

  • On the very first line, enter your OpenVPN username.
  • On the second line, enter your OpenVPN password.

Important: There should be no extra lines or spaces. The file should only contain these two lines.

4

Import Profile and Connect

First, import the profile that matches your primary location (where you'll work most often).

  • If you are mostly off-campus, import OpenVPN.ovpn.
  • If you are mostly on-campus, import OpenVPN-onCampus.ovpn.

After importing, follow these connection steps:

  1. Verify Username: The app should display your correct username. This confirms it has found and read your secret file.
    • If it does not show your username, double-check that your .ovpn and secret files are in the same folder and that the content of the secret file is correct.
  2. Save Password: Check the box labeled "Save Private Key Password".
  3. Enter Password: When prompted, enter the private key password: abnormal.
  4. Connect: Click to connect. If you anticipate working from both locations, I recommend importing the second profile now as well. Follow the same steps to save the private key password so you're prepared to connect from anywhere.
5

Access Your Proxmox Hypervisor

Once you are connected to the VPN, you can access your personal Proxmox hypervisor. Your site is only accessible while the VPN is active.

  1. Visit your site: Open a web browser and enter your unique URL, which will be in the format: http://10.255.254.xx:8006 (where xx is the last octet of your ip address: 21-30).
  2. Use your credentials: Log in with the username root and the unique, complex password provided in the credentials email.
