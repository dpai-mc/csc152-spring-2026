What is Proxmox?

Proxmox Virtual Environment (Proxmox VE) is an open-source virtualization platform that allows you to create and manage virtual machines and containers. Think of it as your personal data center in the cloud.

Your Proxmox Server Includes:

  • Web-based management interface - Access from any web browser
  • Ability to create multiple virtual machines - Run Linux, Windows Server, and other OS simultaneously
  • Virtual networking capabilities - Create networks, VLANs, firewalls
  • Storage management - Configure RAID, LVM, and other storage solutions
  • Snapshot and backup capabilities - Save and restore VM states
  • 24/7 availability - Access your lab environment anytime, anywhere

What You'll Build in Proxmox

Throughout the semester, you will create and configure:

  • Multiple Linux server VMs (Ubuntu Server, Rocky Linux)
  • Multiple Windows Server VMs
  • Virtual networks and subnets
  • Firewall appliances
  • Directory service servers (Active Directory, OpenLDAP)
  • Storage servers with RAID configurations
  • Backup and disaster recovery systems
