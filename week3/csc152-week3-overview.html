Week 3 marks a significant transition from hardware fundamentals to operating system administration by introducing Linux server environments. Students will install their first Linux server operating system, learn to navigate the Linux file system hierarchy, master basic command-line operations, and understand package management fundamentals. This week establishes the foundation for all future Linux administration topics and emphasizes the importance of command-line proficiency in server management.
Key Focus: Unlike desktop Linux distributions, server editions are typically managed entirely through the command-line interface, making CLI competency essential for success in this course and in professional server administration roles.

  • Understanding dependencies and repositories
