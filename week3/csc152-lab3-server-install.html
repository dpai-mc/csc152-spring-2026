Lab 3: Linux Server Installation & Basic Administration

CSC 152 - Network Operating Systems and Administration | Week 3

Due Date: Sunday, February 9, 2026, 11:59 PM

Lab Overview

This lab is your first hands-on experience with Linux server administration. You will install a Linux server operating system in your virtual environment, perform essential post-installation configurations, and begin interacting with the file system and package manager using command-line tools. This foundational lab is crucial for building your proficiency in managing Linux servers.

Learning Objectives:

  • Install a Linux server operating system (Ubuntu Server or Rocky Linux) in a virtual environment.
  • Perform initial post-installation configuration tasks.
  • Navigate the Linux file system hierarchy using command-line tools.
  • Use basic package management commands to install, update, and remove software.
  • Create comprehensive technical documentation for server administration tasks.

Prerequisites:

Before starting this lab, ensure you have successfully accessed your personal Proxmox hypervisor and reviewed all Week 2 learning modules. You should be familiar with VM resource allocation and basic server hardware concepts.

Your Proxmox Resources:

Each student has been provisioned with a personal Proxmox Virtual Environment (PVE) containing approximately 6 vCPUs, 24 GB RAM, and 200 GB HDD total storage capacity for all your VMs. Plan your VM allocations carefully to stay within these total resource limits! For this lab, a single Linux server VM will suffice.

  • Recommended for Linux VM: 2 vCPUs, 2-4 GB RAM, 20-30 GB Storage.

Installation Process Documentation (4 points)

Post-Installation Configuration (2 points)

After the first boot, perform and document these initial configurations:

  • System Updates: Document running system updates with a screenshot showing the process completion. 
    # For Ubuntu:
sudo apt update && sudo apt upgrade
# For Rocky Linux:
sudo dnf update
  • SSH Service Verification: Verify the SSH service is running with a screenshot. 
    sudo systemctl status sshd
  • Network Connectivity Verification: Verify network connectivity with a screenshot of successful tests. 
    ip addr show
ping -c 4 8.8.8.8

Command-Line Navigation Practice (3 points)

Document completing the following tasks in your VM, including commands used and screenshots of verification:

  1. Create Directory Structure: Create the following nested directory structure in your home directory (~/): 
    ~/csc152/
├── week3/
│   ├── labs/
│   ├── notes/
│   └── configs/
└── backups/
    Show commands used (e.g., mkdir -p) and the resulting structure (e.g., using tree or ls -R).
  2. Create Text Files: Create three empty text files in ~/csc152/week3/notes/: installation.txt, commands.txt, questions.txt. Briefly add some content to installation.txt (e.g., "My Linux installation notes"). Show file creation (touch) and verification (ls).
  3. Copy File: Copy commands.txt from ~/csc152/week3/notes/ to ~/csc152/backups/. Show the copy command and verification (ls in both locations).
  4. View Log Files: Use cat, head, and tail to view different system log files (e.g., /var/log/syslog for Ubuntu or /var/log/messages for Rocky Linux). Include a screenshot showing the output of one of these commands. 
    sudo cat /var/log/syslog     # Ubuntu
sudo cat /var/log/messages   # Rocky Linux

File Information and Analysis (3 points)

Run the following commands and document their outputs with screenshots. Then, provide a brief analysis.

  • Disk Usage of Home Directory: 
    du -sh ~/*
  • Filesystem Usage: 
    df -h
  • Find Largest Files in /var (requires sudo): 
    sudo du -ah /var | sort -rh | head -20

Brief Analysis: In 100-150 words, explain what these commands tell you about disk space usage. Which directories or files seem to use the most space? Why might this be important for server administration?

Installing New Packages (4 points)

Install the following useful administrative tools using your distribution's package manager. Include a screenshot showing the installation process completion for one of them.

  • htop (interactive process viewer)
  • tree (directory listing in tree format)
  • net-tools (legacy networking utilities like ifconfig)
  • vim (powerful text editor)
# For Ubuntu:
sudo apt install htop tree net-tools vim
# For Rocky Linux:
sudo dnf install htop tree net-tools vim

For each package installed, document its name, purpose/what it does, and provide a basic usage example with a screenshot (e.g., run htop and capture a screenshot showing system resources).
