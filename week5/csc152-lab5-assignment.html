üêß Part A: Bash Scripting 12 points
system_info.sh 4 points

System Information Script - Create a script that displays comprehensive system information.

  • Display hostname, OS version, kernel version
  • Display uptime and current date/time
  • Display currently logged-in users
  • Display CPU model and core count
  • Display total and available memory
  • Display disk usage for all mounted filesystems
  • Format output in organized sections with headers
  • Save output to ~/system_info_$(date +%Y%m%d_%H%M%S).log
  • Include script in lab report with comments
  • Include screenshot of script execution and output
create_user.sh 4 points

User Creation Script - Create a script that automates user account creation with validation.

  • Accept username as first argument
  • Validate username (alphanumeric only, 3-16 characters)
  • Check if user already exists
  • Create user with home directory
  • Set initial password (prompt securely)
  • Add user to users group
  • Log action to /var/log/user_creation.log (requires sudo)
  • Display success message with user details
  • Include proper error handling
  • Include script in lab report with comments
üì∏ Required Screenshots: Successful user creation, error handling (attempting to create existing user), log file entry
backup_directory.sh 4 points

Backup Script - Create a script that backs up directories with timestamps and logging.

  • Accept source directory and backup destination as arguments
  • Validate that source directory exists
  • Create backup destination if it doesn't exist
  • Create tar.gz archive with timestamp: backup_YYYYMMDD_HHMMSS.tar.gz
  • Display backup progress
  • Verify backup was created successfully
  • Display backup file size and location
  • Log backup details to ~/backup.log
  • Include usage instructions in script comments
  • Include script in lab report
üì∏ Required Screenshots: Script execution with valid arguments, created backup file details, log file contents

  • Validate username format (alphanumeric, 3-16 characters)
